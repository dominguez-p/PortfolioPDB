# Checklist de publicación

## Antes de escribir

- [ ] Definir slug sin acentos ni espacios
- [ ] Crear carpeta del artículo
- [ ] Copiar plantilla base

## Antes de publicar

- [ ] Revisar `<title>`
- [ ] Revisar `meta description`
- [ ] Revisar título visible
- [ ] Revisar fecha
- [ ] Revisar tiempo de lectura
- [ ] Revisar resumen
- [ ] Revisar enlaces internos
- [ ] Revisar ortografía
- [ ] Revisar visualización en móvil
- [ ] Revisar consola sin errores

## Publicación

- [ ] Añadir artículo a `/articles/index.html`
- [ ] Añadir artículo a la home si aplica
- [ ] Commit
- [ ] Push
- [ ] Verificar en GitHub Pages
